Usage ​

CSS 媒体查询主要为 vscode 用户片段。

其中，w 和 ww 为 CSS 预设会考虑支持的断点前缀 （使用方法 px-2 w:px-4 ww:px-6 表示 padding 会根据设备宽度逐渐变大），具体能不能支持，需参照接下来每个页面的介绍。

由于 w 作为单个字符已经被使用，“0.5 w” 用 v 表示。

响应式断点多了生成的CSS体积会可观地增长。需要合理限制数量。
