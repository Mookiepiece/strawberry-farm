# 尺寸 Sizing

## 间距尺寸

::: details 实现在 CSS 一节的间距。

实现仿 Bootstrap 的 4 的整数倍的好处是因为 4 + 8 = 12，可以通过加法得到相同的长度。但是这里我不选 4 的整数倍。

:::

| Custom Property | Unit |
| --------------- | ---- |
| `--0`           | 0    |
| `--1`           | 1px  |
| `--2`           | 2px  |
| `--3`           | 5px  |
| `--4`           | 10px |
| `--5`           | 20px |
| `--6`           | 30px |
| `--7`           | 40px |
| `--8`           | 50px |
