## Intro

本章节介绍文档所使用的元素。

### 渐变背景色（Custom Properties）

```html
<div style="width:150px; height:100px; background: var(--落霞)"></div>
<div style="width:150px; height:100px; background: var(--幻迷)"></div>
```
