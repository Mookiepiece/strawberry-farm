## Spin 加载符

Spin 用于显示加载符。Spin.Container 用于锁定一块加载区域。

### 接口索引

| Spin Property | Description                    | Type      |
| ------------- | ------------------------------ | --------- |
| loading       | 是否显示，不提供默认是一直显示 | `boolean` |
| weight        | 线宽，默认 2                   | `1/2/3`   |
| lazy          | 延迟显示的时间                 | `number`  |
