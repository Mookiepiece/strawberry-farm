## Text Align

Inspired by Github Markdown Extension.

| Class  | CSS                  |
| ------ | -------------------- |
| `[:-]` | `text-align: start`  |
| `[--]` | `text-align: center` |
| `[-:]` | `text-align: end`    |

```html
<div class="[:-]">[:-]</div>
<div class="[--]">[--]</div>
<div class="[-:]">[-:]</div>
```
