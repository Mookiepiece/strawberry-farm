### API

| Property   | Description | Type                | Default | Required |
| ---------- | ----------- | ------------------- | ------- | -------- |
| haircut    |             | `boolean`           | `false` |          |
| children   | description | `string`            |         |          |
| width      |             | `string` / `number` |         |          |
| height     |             | `string` / `number` |         |          |
| background |             | `string`            |         |          |
