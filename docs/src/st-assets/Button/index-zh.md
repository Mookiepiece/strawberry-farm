### 接口索引

| Button Property | Description  | Type      |
| --------------- | ------------ | --------- |
| primary         | 主要按钮样式 | `boolean` |
| textual         | 文字按钮样式 | `boolean` |
| block           | 块样式       | `boolean` |

\*除上述属性，按钮仍继承原生 `<button>` 属性
